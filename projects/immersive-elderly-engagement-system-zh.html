重點項目

沉浸式長者互動系統（XR）

一個服務學習部署項目：以 Unity + Joy-Con 互動設計為核心，結合 360° 虛擬步行與 photogrammetry／點雲重建，促進長者參與、回憶與數碼共融。

GitHub 專案 可應要求提供示範／材料 返回項目
重點：本項目並非單純追求「炫技 XR」，而是以長者可快速學習的操作為前提，透過熟悉場景與具文化意義的內容，促進對話、共鳴與身心福祉。
註：由於資料量及現場私隱等限制，公開 repo 未必包含所有影片／3D 資產。

摘要（CV／海報版本）

360° 虛擬步行 + 3D 重建 + 無障礙互動設計。

本項目為雲南弱勢長者設計並部署沉浸式數碼體驗，整合以下三個要素：

  • 360° 虛擬步行：以 360 影片方式探索具文化意義之地點。
  • 高擬真 3D 重建：以 photogrammetry／點雲資產（例如 .ply）重建熟悉環境。
  • Unity + Joy-Con 互動：以簡單可重複的操控支援導覽與動作回饋。

於現場部署與使用者測試期間，我們觀察到參與者的情緒投入、懷舊感及社交互動有所提升，顯示沉浸式技術可在適切的互動設計與人本服務安排下，支援長者身心福祉、記憶連結及數碼共融。

系統概覽

以「低數碼素養使用者」為主要設計對象。

使用流程

  1. 主選單：瀏覽多個目的地（每個目的地對應一段影片／場景）。
  2. 體驗場景：播放所選 360° 虛擬步行內容，並顯示 UI 疊加資訊（進度、路程、步數）。
  3. 設定面板：暫停／繼續、返回主選單，以及調整敏感度。

互動設計目標

  • 可學習性：以簡單按鍵與搖桿為主，跨場景操作一致。
  • 動作回饋：可按 IMU 動作幅度控制播放（例如活動不足時自動暫停）。
  • 無障礙：強調可讀性與降低認知負擔（大字體、簡化導覽）。

控制方式（Joy-Con）

以易學、低出錯率為設計原則。

情境 輸入 動作
主選單 方向鍵 左／右 上一個／下一個目的地
主選單 肩鍵（確認） 進入體驗場景並載入所選內容
體驗 搖桿 旋轉視角（yaw／pitch）
體驗 IMU 動作（陀螺儀幅度） 按活動閾值自動播放／暫停
體驗 肩鍵 開啟設定面板（並暫停影片）
設定面板 方向鍵 下 關閉設定面板（並恢復播放）
設定面板 方向鍵 上 返回主選單

開發期間可使用滑鼠作為後備輸入；部署以 Joy-Con 為主。

內容製作流程

拍攝 → 重建 → Unity 整合。

  • 360° 拍攝：以 Insta360 進行現場 360 拍攝；如條件允許，加入航拍素材作補充。
  • 3D 重建：以 photogrammetry 工具（例如 LumaAI）產出重建模型與點雲。
  • Unity 整合：將 360 內容、UI 疊加與互動邏輯整合成可運行應用。
  • 進階渲染（可擴展）：未來可研究以點基渲染／splatting 類方法提升 3D 呈現品質。

我們刻意把使用者端體驗保持簡潔，同時預留路徑讓系統可在硬件能力與部署限制允許時，逐步納入更高擬真的 3D 資產。

現場部署與反思

服務學習現場觀察之定性成果。

觀察到的成果

  • 初期抗拒與猶疑可在短時間適應後轉化為好奇與愉悅。
  • 當參與者辨認到熟悉環境時，會觸發明顯的懷舊與情緒投入（強烈的互動驅動因素）。
  • 體驗常自然轉為社交活動：使用者會邊探索邊分享故事與回憶。

主要挑戰

  • 系統穩定性：偶發卡頓／同步問題會中斷沉浸感。
  • 溝通障礙：數碼素養差異大，需更耐心且更直觀的引導。
  • 現場限制：空間、噪音與設備安排影響流程節奏與可靠性。

主題句

以同理心為本的科技，配合真誠的人際互動與協作，能透過縮窄數碼落差，為受壓抑的精神帶來支持與重建。

限制與後續方向

  • 輸入一致性：統一 axis 映射，並評估遷移至 Unity 新 Input System 的成本與收益。
  • 媒體規模化：大量影片可能令 build 變大；可考慮 Addressables／AssetBundles。
  • 步數偵測：以濾波 + peak detection 取代簡單閾值，並加入裝置校準流程。
  • Joy-Con 可靠性：改善連線狀態提示、斷線處理，以及控制器選擇。
  • 無障礙迭代：更早進行共同設計（co-design）與可用性測試，持續簡化介面與導覽。

如何運行（高層次）

求職用途：以簡短、可重現的描述呈現。

  1. 以相容的 Unity Editor 版本開啟專案（Unity 6／或 2022 LTS，視分支與工具而定）。
  2. 載入主選單場景並按 Play。
  3. 於 Windows 透過藍牙配對 Joy-Con，並確認輸入被正確辨識。
  4. 選擇目的地以開始 360° 體驗。

如需要更完整的 build／媒體與部署筆記，可聯絡我安排提供。
