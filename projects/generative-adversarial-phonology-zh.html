重點項目

生成對抗音系建模 — ciwGAN：越南語元音長度對立

以 conditional WGAN-GP 在固定尺寸 log-mel spectrogram（128×128）上生成越南語元音音訊，並以客觀語音指標 （VOT、強度）評估短／長元音的可控性與語音逼真度。

研究問題：在沒有明確標註時間線索（如 burst 與 voicing onset）的前提下，conditional GAN 是否能同時做到： （1）短／長元音可控；（2）維持具語音學意義的微秒級時間結構（尤其是 VOT）。

概覽

最後更新：2026-02-04

本項目提供一個可重現的研究型工作流程：訓練 conditional WGAN-GP（ciwGAN）以生成越南語元音之 log-mel spectrogram patch， 再以 Griffin–Lim 風格的 mel inversion 還原音訊。評估重點不是僅靠主觀聆聽，而是使用可量化、具語音學意義的客觀指標： VOT（Voice Onset Time） 與 強度（RMS dB）。

  • 模型：conditional WGAN-GP（critic + duration class head）。
  • 條件：短／長元音類別 $c \in \{0,1\}$，以 one-hot 與 latent $z$ 串接。
  • 表示：固定尺寸 log-mel patch，正規化至 $[-1,1]$。
  • 評估：VOT（burst→voicing 延遲）、強度統計（mean/median/max dB、peak-to-mean）。

方法（實作摘要）

以固定尺寸 spectrogram 建模，並建立評估與打包流程。

資料與特徵

  • 重採樣至 16 kHz；mel spectrogram（n_mels=128, n_fft=1024, hop_length=256）。
  • Power→dB；將 [-80,0] 映射至 [-1,1]。
  • 時間維固定為 128 frames（不足補 -1；過長隨機裁切）。

模型與損失

  • Generator：dense 投影 → upsample conv blocks → tanh 輸出。
  • Critic：conv downsample；兩個 head：critic score + 2 類 duration classifier。
  • 目標：WGAN-GP + gradient penalty（λ=10）+ 輔助分類損失（real 與 fake）。
  • 訓練：critic_steps=5；Adam lr=2e-4, β1=0.5, β2=0.9。

音訊重建（inversion）

以 mel_to_audio（Griffin–Lim 風格）將 spectrogram 還原為 waveform。其優點是本機可重現且依賴少，但對幅度與相位逼真度 有明顯限制，亦直接反映於強度指標的偏移。

解讀（以現有 artifacts 為準）

  • Long-vowel VOT 中位數隨訓練提升，並在 100 epochs 與已存 real reference 中位數完全一致（0 ms 誤差）。
  • Short-vowel VOT 由 pilot 大幅改善，但在 100 epochs 仍有 +7.5 ms 的中位數偏移。
  • 強度受 Griffin–Lim inversion 影響顯著；normalisation 後仍保留約 15–18 dB 的差距。
重要限制：real VOT reference 覆蓋不足

目前 repository 內的 real VOT CSV 在部分比較中顯示 real_n = 1，因此無法可靠估計 real distribution 的 mean/std。 建議在具備完整真實音訊資料的環境中重新生成 real VOT 參考，並重跑比較與繪圖。

限制與後續工作

  • 補齊 real VOT reference：以完整真實資料重建 runs/vot_real.csv，再重跑 compare 與 plots。
  • 處理資料不平衡：若 long:short 比例失衡，採用 sampling/weighting 重新訓練以觀察 short-vowel VOT 是否改善。
  • latent probing：完成 latent interpolation + 指標計算，檢驗長度線索主要由 class conditioning 或 latent space 控制。
  • 引入 vocoder：以 HiFi-GAN 等 neural vocoder 取代 Griffin–Lim，改善幅度與相位逼真度。
  • 量測穩健性：針對 VOT heuristic 在 synthetic 音訊上的敏感度做 sanity checks 與人工抽查。
