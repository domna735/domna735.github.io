即時面部表情辨識系統（Real-Time FER）

PyTorch · OpenCV · ONNX Runtime · MobileNetV3 · YuNet · CLAHE · KD/DKD

  • 建立 7 類表情辨識流程（pipeline），提升少數類別（Disgust、Fear、Sad）之辨識表現。
  • 設計教師模型—學生模型蒸餾（Teacher–Student；RN18／B3／CNXT → MobileNetV3），Macro-F1 ≈ 0.79，驗證集準確率約 80%。
  • 建置多執行緒即時推論流程（擷取 → 前處理 → 推論 → 平滑），CPU 可達 25–30 FPS。
  • 透過時間序列穩定化（EMA、hysteresis、投票視窗）降低輸出抖動（flicker），提升使用體驗。
  • 提供可重現之機器學習流程：資料清理 → 清單（manifest）驗證 → 訓練 → 匯出 → 部署。

多模態虛擬溝通系統（MVP）

Python · ONNX Runtime · Azure OpenAI（選用）· Azure Speech（選用）· Windows TTS · DirectML

  • 由零開始建立可即時運行的多模態互動原型，整合表情辨識、對話生成與語音輸出至單一應用。
  • 以離線規則式代理為預設，並提供 Azure OpenAI 後端（選用）；支援 persona 切換，回應可依情緒與 persona 調整。
  • 採用 Windows 內建 TTS 作零設定可靠輸出，並支援 Azure Speech TTS（選用）與線上服務失效時自動降級（fallback）。
  • 語音輸入（STT，選用）：push-to-talk 及免手持連續模式，並在 TTS 播放期間自動暫停收音以提升對話自然度。
  • 加入語調特徵（pitch、能量）分析（選用）、JSONL 會話日誌與延遲統計腳本，便於評估、可重現與除錯。

生成對抗音系建模（Generative Adversarial Phonology）

TensorFlow · WGAN-GP · SpecGAN · 語音生成

  • 將既有 TF1 程式碼重構至 TF2 tf.keras 訓練流程，提升可重現性與穩定度。
  • 實作 WGAN-GP 損失函數並優化上採樣（upsampling）層，改善收斂與語音合成品質。
  • 建立 SpecGAN 系統，支援可控的 vowel length contrasts 生成。
  • 曾為 UC Berkeley Annual Phonology Meeting（2025）相關工作提供技術貢獻。

Unity · Insta360 · LumaAI · Photogrammetry · 互動設計

  • 為雲南弱勢長者設計沉浸式體驗，結合 360° 虛擬步行與高擬真 3D 重建。
