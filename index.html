Real-Time Facial Expression Recognition System

PyTorch · OpenCV · ONNX Runtime · MobileNetV3 · YuNet · CLAHE · KD/DKD

  • Engineered a 7-class FER pipeline boosting minority-class recognition (Disgust, Fear, Sad).
  • Built teacher–student distillation (RN18/B3/CNXT → MobileNetV3) achieving Macro-F1 ≈ 0.79 and ~80% accuracy.
  • Developed multi-threaded real-time inference (async capture → preprocessing → inference → smoothing) at 25–30 FPS on CPU.
  • Reduced flicker via temporal stabilisation (EMA smoothing, hysteresis, voting window).
  • Delivered a reproducible pipeline: dataset cleaning → manifest validation → training → export → deployment.
